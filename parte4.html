Aplicando un forms a una página web

Paso 1: Agregar Bootstrap al proyecto


Se puede agregar ya sea descargando los archivos css, js y colocándolos donde se encuentra el proyecto, o referenciándolos en el head y el body de tu proyecto.


Manual para descargar Bootstrap en tu proyecto

Descargar manual

Paso 2: Hacer el formulario en el HTML del proyecto


Existen muchas opciones y elementos para colocar en un formulario, asi que es recomendable ver los componentes que sean más favorables para tu sitio web


Estos son algunos de los elementos más utilizados.

Importante mencionar que el formulario tiene su propia etiqueta <form></form>


Correo electrónico y contraseña

<div class="mb-3">
<label for="exampleInputEmail1" class="form-label">Dirección de correo electrónico</label>
<input type="email" class="form-control" id="exampleInputEmail1" aria-describedby="emailHelp">
</div>

<div class="mb-3">
<label for="exampleInputPassword1" class="form-label">Contraseña</label>
<input type="password" class="form-control" id="exampleInputPassword1">
</div>

Entrada de Archivo y Listas de datos

<div class="mb-3">
<label for="formfile" class="form-label">Por favor seleccione un archivo</label>
<input class="form-control" type="file" id="formFile">
</div>

<label for="exampleDataList" class="form-label">Pronvincias de Costa Rica</label>
<input class="form-control" list="datalistOptions" id="exampleDataList">
<datalist id="datalistOptions">
<option value="Heredia">
<option value="San José">
<option value="Alajuela">
<option value="Cartago">
<option value="Limón">
<option value="Puntarenas">
<option value="Guanacaste">
</datalist>

Otros elementos


Código para un checkbox

<div class="mb-3 form-check">
<input type="checkbox" class="form-check-input" id="exampleCheck1">
<label class="form-check-label" for="exampleCheck1">Echáme un vistazo</label>
</div>

Código para un botón

<button type="submit" class="btn btn-primary">Enviar</button>

  • Para desabilitar algun elemento se usa la palabra disabled
  • Para desabilitar un conjunto de elementos se puede usar la etiqueta <fieldset disabled="disabled"></fieldset>

<input class="form-control" id="disabledInput" type="text" placeholder="Elemento desabilitado" disabled>
<input class="form-control" type="text" value="Control de solo lectura" aria-label="readonly input example" readonly>

Atributos y clases que se pueden usar en los diferentes componentes

  • Placeholder
  • form-control-lg/sm
  • form-text
  • form-control-plaintext
