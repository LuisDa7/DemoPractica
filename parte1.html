¿Cómo implementar un dropdown de Bootstrap en tu proyecto?

Paso 1: Incluir Bootstrap en tu proyecto


Para usar Bootstrap, primero necesitas incluir los archivos CSS y JavaScript de Bootstrap en tu proyecto. Puedes hacerlo añadiendo las siguientes líneas en el head y justo antes del cierre del body en tu archivo HTML.


CSS de Bootstrap:


<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous">


Incluir JavaScript de Bootstrap y dependencias Bootstrap:


<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"
    integrity="sha384-YvpcrYf0tY3lHB60NNkmXc5s9fDVZLESaAA55NDzOxhy9GkcIdslK1eN7N6jIeHz"
    crossorigin="anonymous"></script>


Paso 2: Agregar el dropdown en tu HTML


Añade el siguiente código HTML en el lugar donde quieras que aparezca tu dropdown:


Código HTML:


<div class="dropdown">
    <button class="btn btn-secondary dropdown-toggle" type="button" data-bs-toggle="dropdown" aria-expanded="false">
        Dropdown button
    </button>
    <ul class="dropdown-menu">
        <li><a class="dropdown-item" href="#">Action</a></li>
        <li><a class="dropdown-item" href="#">Another action</a></li>
        <li><a class="dropdown-item" href="#">Something else here</a></li>
    </ul>
</div>

Paso 3: Personalizar el dropdown


Puedes personalizar el dropdown cambiando las clases y el contenido según tus necesidades.
