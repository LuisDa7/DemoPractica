¿Cómo implementar Bootstrap en tu proyecto?


Paso 1

Para usar Bootstrap, primero necesitas incluir los archivos CSS y JavaScript de Bootstrap en tu proyecto. Puedes hacerlo añadiendo las siguientes líneas en el head y justo antes del cierre del body en tu archivo HTML.


CSS de Bootstrap:


<link href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css" rel="stylesheet">


Incluir JavaScript de Bootstrap y dependencias Bootstrap:


<script src="https://code.jquery.com/jquery-3.5.1.slim.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.9.2/dist/umd/popper.min.js"></script>
<script src="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>
