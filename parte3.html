Modales

El modal es una ventana emergente que se superpone sobre cualquier contenido principal de una página. Aunque es bastante flexible y se le puede dar usos muy variados, normalmente se utiliza para dar mensajes breves o funcionan como medio de confirmación antes de efectuar una acción.

Clases para personalizar tu modal:

  • Clase para centrar el modal: "modal-dialog-centered".

  • Clase para aumentar el tamaño del modal: "modal-xl".

  • Clase para disminuir el tamaño del modal: "modal-sm".

  • Clase para agregarle una barra de desplazamiento al modal en caso de que el cuerpo del modal sea muy extenso: "modal-dialog-scrollable".

Pagination

Son diferentes particiones de información con contenido similar. Son utilizadas cuando se vuelve muy pesado o molesto para el usuario si toda la información se encontrase en una única página principal. La cantidad de información que contiene cada paginación depende exclusivamente de las desiciones del desarrollador.

Clases para personalizar nuestra paginacion:

  • Activar o desactivar una de las particiones. Clases: ".active .disabled". Estas clases se agregan al elemento de la lista que represente la partición que queremos activar o desactivar.

  • Alterar el tamaño de la paginación. Clases: ".pagination-lg .pagination-sm".

  • Cambiar la alineación con las utilidades de flexbox. Clases: ".justify-content-center .justify-content-end".

Navs & Tabs

"Navs" es una abreviatura de navegación. Los navs de Bootstrap nos permiten crear menús de navegación flexibles y nos ofrece diversas clases para darles el estilo deseado

Clases para personalizar nuestros navs:

  • Clases para cambiar la alineación de nuestro nav utilizando las utilidades de flexbox:

    • Clase para centrar el menú de navegación: ".justify-content-center".

    • Clase para alinearlo a la derecha: ".justify-content-end".

  • Clase para crear regiones tabbables: ".nav-tabs".

  • Clase para ordenar el nav verticalmente: ".flex-column".

  • Clase para separar los elementos proporcionalmente: ".nav-fill".

  • Clase para crear un menú de navegación responsivo utilizando funciones flexbox: "flex-column flex-sm-row".
